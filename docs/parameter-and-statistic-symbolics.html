3.11 Parameter and statistic - symbolics

(P) (S)
expected value (mean): \(E(X), \mu\) sample mean: \(\bar{x}\)
proportion: sample proportion: \(\hat{p}\)
standard deviation: \(\sigma\) sample standard deviation: \(S, S(X), s\)
variance: \(\sigma^2\), \(Var(X)\) sample variance: \(S^2, S^2(X), s^2\)
