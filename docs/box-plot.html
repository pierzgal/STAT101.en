3.21 Box plot

Like bar plots and histograms, the box plot is used to show the distribution of the data.

This type of plot illustrates the basic characteristics of a data distribution, and is often used when researchers want to see how the distribution of a quantitative variable is affected by some qualitative variable (or discrete quantitative) (e.g .: life expectancy vs. continent).
